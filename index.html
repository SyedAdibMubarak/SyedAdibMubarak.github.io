Ghost of Sparta

This is a game created for our game development class using Construct 2 game engine. It is a simple 2D shooter platformer aimed to educate about mosquitos in a fun way, albeit quite violent. The game engine is quite simple and limited, it is definitely quicker to develop games using its ready made features and it allows tweaking some variables. I experimented with its Sin function to create a variety of enemy movements. It is definitely a fun project with a lot of Pop-culture references
